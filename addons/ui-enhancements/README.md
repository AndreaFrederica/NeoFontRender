# NFR UI Enhancements for Minecraft 1.7.10

Optional client addon providing modern tooltips, smooth list scrolling, native
text cursors, configurable screen effects, enhanced chat history, and extensible
HUD status bars. The HUD supports Classic, Modern, Flat, Glass, Segmented, and
Minimal themes, and uses AppleCore's public API for food previews and exhaustion
when AppleCore is installed. It requires Neo Font Render 0.3.5 or newer and
MixinBooter.
